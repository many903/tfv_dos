OCR TO EXCEL APP v2.0
======================

DESCRIPCIÓN:
Aplicación de escritorio para extraer texto de imágenes usando OCR
y exportarlo a Excel. Funciona 100% offline.

CARACTERÍSTICAS:
✅ Extracción de texto de imágenes (JPG, PNG, BMP, TIFF)
✅ Reconocimiento de tablas y datos estructurados
✅ Preprocesamiento de imágenes para mejor precisión
✅ Edición de datos extraídos
✅ Exportación a Excel (.xlsx) y CSV
✅ Interfaz gráfica intuitiva
✅ 100% OFFLINE - No requiere internet
✅ Multi-idioma (Español, Inglés, etc.)

INSTALACIÓN RÁPIDA:
1. Ejecuta "Instalar_OCR.exe" como administrador
2. Sigue los pasos del instalador
3. ¡Listo! La aplicación estará lista para usar

INSTALACIÓN MANUAL:
1. Instala Python 3.8+ desde python.org
2. Ejecuta: pip install -r requirements.txt
3. Instala Tesseract OCR (ver instrucciones abajo)
4. Ejecuta: python OCR_APP.py

INSTALAR TESSERACT OCR:
- Windows: Descarga desde https://github.com/UB-Mannheim/tesseract/wiki
- Linux: sudo apt install tesseract-ocr tesseract-ocr-spa
- Mac: brew install tesseract

USO:
1. Abre la aplicación
2. Carga una imagen (Archivo → Abrir Imagen)
3. Ajusta preprocesamiento si es necesario
4. Haz clic en "Procesar OCR"
5. Revisa y edita los datos
6. Exporta a Excel (Archivo → Exportar)

ATAJOS DE TECLADO:
Ctrl+O  - Abrir imagen
Ctrl+E  - Exportar a Excel
Ctrl+S  - Guardar texto
F1      - Ayuda
